# 智慧教育刷课脚本

## 简介

本项目是一个自动化脚本，用于辅助用户在智慧教育平台上完成课程学习。该脚本通过模拟用户操作来自动播放课程视频和阅读课件。

## 使用前提

- 确保你已经安装了Chrome浏览器。
- 确保你已经登录智慧教育平台。

## 使用方法

### 1. 下载可执行文件

请从以下链接下载刷课脚本的可执行文件：

[下载刷课脚本](#)

### 2. 运行可执行文件

双击下载的`.exe`文件即可运行。

### 3. 脚本操作

- 脚本会自动打开Chrome浏览器并登录智慧教育平台。
- 脚本将自动完成课程视频的播放和课件的阅读。
