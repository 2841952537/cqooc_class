## 简介

项目分为两个版本

1. 高等教育平台_Chrome.exe

2. 高等教育平台_Requests.exe

   Chrome版本基于DrissionPage库开发，是自动化程序（需要浏览器支持）

   Requests版本基于requests库开发，是协议程序（不需要浏览器支持）

## 使用前提

- Chrome版本确保你已经安装了Chrome浏览器。
- 拥有重庆智慧教育平台账号。
